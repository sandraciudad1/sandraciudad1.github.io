Información del proyecto

  • Nombre del proyecto: Torneo de Tenis Femenino
  • Fecha realización: Abril 2022
  • Objetivo: Aprender a manejar bases de datos mediante el desarrollo de una aplicación de escritorio.

Detalles del proyecto

El sistema a desarrollar consiste en una aplicación que simula partidos y campeonatos de tenis femenino. La base de datos que se gestiona contiene una serie de tablas donde se almacena la información sobre las distintas ediciones de los torneos, los partidos, las jugadoras y sus paises.
