Descripción del proyecto


Este proyecto consistió en el desarrollo de una aplicación de realidad aumentada en Unity utilizando el SDK de Vuforia, centrada en la detección y gestión de múltiples image targets de forma simultánea. El objetivo fue diseñar una experiencia interactiva que reaccione de manera diferente en función de la combinación de targets detectados por la cámara del dispositivo, explorando así el potencial del reconocimiento de múltiples marcadores en un mismo espacio físico.

Cada imagen target fue asociada a la aparición de un modelo 3D. Adicionalmente, se definieron combinaciones específicas de dos o tres targets que, al ser detectadas de forma conjunta, desencadenaban una respuesta integrada, como la fusión de modelos y la transformación de objetos en escena.

La lógica de detección se implementó mediante ObserverBehaviour y el uso de eventos OnTargetFound y OnTargetLost, gestionando la sincronización entre múltiples observadores mediante scripts personalizados que coordinan el estado activo de cada target. Se utilizó también ARCamera como componente central de seguimiento, con ajustes optimizados para garantizar estabilidad y precisión en dispositivos móviles.

Todo el proyecto se estructuró bajo un enfoque modular y escalable, utilizando programación orientada a objetos en C# para facilitar la gestión de estados, combinaciones válidas y comportamiento de los modelos aumentados. La experiencia final se diseñó para ser intuitiva, educativa y expandible, permitiendo su adaptación a nuevos targets y casos de uso sin alterar la estructura base de la aplicación.
