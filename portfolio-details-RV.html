Descripción del proyecto


Este proyecto consistió en el desarrollo de una experiencia inmersiva en Realidad Virtual simulando un recorrido por un entorno museístico interactivo. Implementado en Unity y orientado al uso con gafas de realidad virtual, el sistema permite al usuario desplazarse entre distintos nodos de observación dentro de un circuito cerrado, donde puede explorar libremente elementos decorativos y objetos expositivos en 3D.

La interacción principal se basa en un sistema de puntero visual fijado al centro de la vista del jugador, que activa una barra radial al mantenerse sobre una esfera de desplazamiento. Dicha barra, desarrollada con componentes de UI y animada mediante el plugin DOTween, gestiona el tiempo de espera hasta completar la acción y desencadenar el desplazamiento hacia otro nodo. El movimiento de la cámara principal se realiza mediante interpolaciones suaves para ofrecer una transición fluida entre puntos del recorrido, mejorando así la experiencia inmersiva.

El sistema está estructurado en base a prefabs reutilizables que encapsulan tanto las esferas interactivas como los comportamientos asociados a la navegación. El entorno 3D fue montado a partir de recursos obtenidos del Unity Asset Store, y toda la lógica de navegación se organizó aplicando principios de programación orientada a objetos en C#, permitiendo un flujo claro y escalable del recorrido.

Este tipo de aplicación no solo resulta útil como experiencia cultural o educativa, sino también como herramienta de visualización arquitectónica o comercial, permitiendo previsualizar espacios antes de ser construidos, con importantes beneficios en términos de toma de decisiones, ahorro de costes y personalización del entorno.
