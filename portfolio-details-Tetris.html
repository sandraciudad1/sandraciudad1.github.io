Descripción del proyecto


Este proyecto consistió en el desarrollo completo de una versión funcional del clásico Tetris. El objetivo fue diseñar un videojuego tipo arcade que aplicase de forma integral los conocimientos adquiridos durante el curso, priorizando una arquitectura escalable y una separación clara de responsabilidades a nivel de scripting.

La lógica de juego se implementó en C# siguiendo una estructura modular basada en el uso de prefabs para cada tipo de tetrominó, controlados mediante un sistema de rotación y caída automatizada. Se empleó una cuadrícula lógica en memoria para gestionar la ocupación de celdas y detectar líneas completas, que se eliminan dinámicamente al ser completadas, desplazando las líneas superiores. Los inputs del jugador se gestionaron mediante eventos de teclado para el desplazamiento lateral, rotación y caída rápida de las piezas.

La escena se construyó con un enfoque 2D utilizando RectTransform y Canvas, y la interfaz de usuario muestra información acerca de la puntuación. El desarrollo se enfocó tanto en la funcionalidad como en la mantenibilidad del código, aplicando principios de programación orientada a objetos y técnicas de refactorización que facilitan futuras ampliaciones del juego.
